Getting Started
===============

Installing Pulserver
--------------------

Pulserver is available on PyPi

.. code-block:: bash

    pip install pulserver

Development Version
~~~~~~~~~~~~~~~~~~~

If you want to modifiy the mrtwin code base

.. code-block:: bash

    git clone https://github.com/INFN-MRI/pulserver
    pip install -e ./pulserver[test, dev, doc]
